Design using CATIA V5

Existing Panel :

1. Start a New Part

  • Open CATIA V5
  • Go to File > New and select Part to create a new part file

2. Create the Panel

  • Go to the Sketcher workbench, and select the XY Plane to start sketching.
  • Draw a rectangle with the length of 1.5 ft and width of 0.75 ft. Convert these to inches (1 ft = 12 inches): Length: 1.5 x 12 = 18 inches Width: 0.75 x 12 = 9inches
  • Exit the sketch
  • Go to the Pad feature and specify the panel's height as 0.175 ft (convert to inches: 0.175 x 12 = 2.1 inches. )
  • Click OK to create the solid panel.

3. Add the Honeycomb Structure To create a honeycomb with a wall thickness of 0.125 inches and a cell size of 0.03 inches:

  • In the Sketcher workbench, select the top face of the panel.
  • Draw a hexagon with a cell size of 0.03 inches (this is the distance across the hexagon).
  • Use the Rectangular Pattern or Circular Pattern tool to replicate the hexagon pattern across the surface of the panel.

  • Use Save As to store the completed design.

Innovative Panel :

1. Start a New Part

  • Open CATIA V5
  • Go to File > New and select Part to create a new part file

2. Create the Base Panel Layer

  • Go to the Sketcher workbench and select the XY Plane
  • Draw a rectangle for the base layer with the required length and width: Length: 1.5 ft (convert to inches: 1.5 x 12=18 inches) Width: 0.75 ft (convert to inches: 0.75 x 12= 9 inches)
  • Exit the sketch
  • In the Part Design workbench, use the Pad feature to extrude the rectangle to the specified height: Height: 0.175 ft (convert to inches: 0.175 x 12=2.1 inches).

3. Duplicate the Layer for a 5-Layer Panel

  • Select the base panel layer created in the previous step
  • Go to Insert > Transformations > Translation to create a copy of the layer
  • In the Translation options: Select Upward (along the Z-axis). Set the distance to 2.1 inches (the height of each layer). Set the Instances to 5 (to create five layers in total).
  • Click OK to apply the transformation.

4. Final Adjustments and Assembly

  • You should now have five identical layers stacked vertically with the specified dimensions.
  • Use Assembly constraints if required to adjust the alignment of the layers precisely
  • Save the completed design by going to File > Save As.
